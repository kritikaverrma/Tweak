# Tweak
It's a twitter clone make using MERN stack using TypeScript.
## Functionality and Features
1. Signup and login using JWT Authentication.
2. Create, Read, Delete Tweets, Like & Unlike Tweets, Tweet with Images/GIFs
3. User can like/unlike and comment on a Post.
4. User can upload a tweet or picture.
5. User can delete only his own Posts.
6. User can receive Notification from others about like, follow, comment.
7. User can see only his followers post and see a suggested User container.
8. Profile Customization- User can edit his own profile (Avatar, Bio, Cover Image)
9. User can see like Posts in their profile.
