# Tweak
It's a twitter clone make using MERN stack using TypeScript.
## Functionality and Features
1. Authentication using JWT
2. User can like/unlike and comment on a Post.
3. User can upload a tweet or picture.
4. User can delete only his own Posts.
5. User can receive Notification from others about like, follow, comment.
6. User can see only his followers post and as see a suggestion User container.
7. User can edit his own profile.
8. User can see other Users like Posts in there profile.
